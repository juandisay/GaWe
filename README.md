# POMODORO WITH SESSION

GaWe is a simple Pomodoro timer desktop app built with Tauri, React, TypeScript, and Vite. It helps users focus by tracking Pomodoro sessions and organizing work into structured blocks, making it easier to manage productivity and breaks. The app is designed to run natively on desktop platforms with a modern, lightweight interface.
